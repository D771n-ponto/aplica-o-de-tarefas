# Gerenciador de Tarefas 

> Uma aplicação Single Page Application (SPA) para gerenciamento de tarefas, desenvolvida para praticar conceitos fundamentais do React como Rotas, Context API e Hooks.


##  Sobre o Projeto:

Este projeto permite que o usuário crie, visualize, edite e exclua tarefas (CRUD). A aplicação foca na **experiência do usuário** e na **persistência de dados**, garantindo que as tarefas não sejam perdidas ao atualizar a página.

### Principais Funcionalidades:

- **Adicionar Tarefa:** Criação de novas tarefas com título.
- **Listar Tarefas:** Visualização de todas as tarefas cadastradas.
- **Editar Tarefa:** Alteração do título de uma tarefa existente.
- **Excluir Tarefa:** Remoção permanente de uma tarefa.
- **Marcar como Concluída:** Toggle visual para tarefas finalizadas.
- **Persistência Local:** Os dados são salvos no `localStorage` do navegador.

---

##  Tecnologias Utilizadas:

- **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces.
- **[React Router Dom (v6)](https://reactrouter.com/)** - Gerenciamento de rotas e navegação.
- **Context API** - Gerenciamento de estado global (Store).
- **Hooks** - `useState`, `useEffect`, `useContext`, `useParams`, `useNavigate`.
- **CSS3** - Estilização responsiva e limpa.
